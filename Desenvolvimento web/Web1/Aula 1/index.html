SISTEMA GERENCIADOR DE INFORMAÇÃO

Desenvolver WEB

O Sistema gerenciador de informação que iremos desenvolver servirá para aprendermos os recursos necessários a um desenvolver WEB. Dessa forma, ao término do cuso, poderemos concorrer a uma vaga no mercado de trabalho dentro da área de Tecnologia da informação (TI). Tudo isso dependerá também da minha dedicação, durante todo o curso.
